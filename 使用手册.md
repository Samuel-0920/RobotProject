# MyRobotProject 使用手册

本文档说明 **MyRobotProject** 目录结构、3+1 架构、环境配置与日常命令。

**外部组件来源**（学长工程、ADM_DP、CLIP 等）见 **[外部来源说明.md](./外部来源说明.md)**，请务必阅读。

运行任何 Python 前，请将工作目录切换到 **`MyRobotProject`**（本文件所在目录）。

---

## 1. 目录结构

| 路径 | 说明 |
|------|------|
| 根目录 `*.py` | 3+1 分层：`perception_engine`、`cognitive_brain`、`motor_adapter`、`test_pipeline`；兼容层 `perception_wrapper`、`vlm_brain` |
| `graduate_pro/` | 视觉检测栈（`vision_ai`、`EnhancedDetectionPipeline`），**来源见外部来源说明** |
| `ADM_DP/` | Diffusion Policy / `LanguageFiLM` 路径，**来源见外部来源说明** |
| `scripts/` | 辅助脚本（如 `download_vision_weights.py`、Windows cuDNN 说明） |
| `outputs/` | 运行产物：`detection_output_*`、`vla_viz_output`、`logs`、`output` 等（从仓库根整理迁入，可按需清理） |
| `requirements-g1-env.txt`、`G1_ENV_SETUP.md` | Conda / 感知环境说明 |
| `README_VLA_SYS.md` | 历史架构说明（若与本文冲突以本文为准） |

**未迁入本目录**（仍在上一级 `code/` 下，属其它仓库或大数据）：

- `FYP_robotics/`、`RealRobot/`、`OCID-dataset/` 等 — 使用请在手册或配置中写清路径，例如 `..\OCID-dataset\...`。

## 2. 架构（3+1）

1. **`perception_engine.py`** — 感知：RGB-D + 位姿 → `EnhancedDetectionPipeline.build_reference_library` → 场景 JSON。不承载本层 CLIP 文本编码。
2. **`cognitive_brain.py`** — 认知：Faster-Whisper、`parse_instruction_with_vlm` / `CognitiveBrain.decide`、失败反思 `reflect_on_failure`。
3. **`motor_adapter.py`** — 运动适配：语义短句 → **ADM_DP 内 `LanguageFiLM` + CLIP** → `[1, 512]` 张量。
4. **`test_pipeline.py`** — 冒烟：Mock RGB-D → JSON → 决策 → 张量形状。

兼容入口：`perception_wrapper.py`、`vlm_brain.py` 转导至上述模块。

---

## 3. 环境与依赖

- **完整感知**：按 `G1_ENV_SETUP.md` 与 `requirements-g1-env.txt` 配置（含 `graduate_pro` 所需权重，见 `scripts/download_vision_weights.py`）。
- **仅认知 + 冒烟**：`numpy`、可选 `openai`；语音 CLI 需 `faster-whisper`、`sounddevice`、`keyboard`、`httpx`。
- **FiLM 向量**：`torch`、CLIP、且 `ADM_DP/policy/Diffusion-Policy` 可被 `motor_adapter` 导入。

**Windows**：`EMBODIED_WHISPER_FORCE_CPU=1` 可强制 Whisper 用 CPU；`cognitive_brain` / `embodied_command_hub` 会尝试把 Conda 的 `Library\bin` 提前加入 PATH。

---

## 4. 环境变量（常用）

| 变量 | 作用 |
|------|------|
| `OPENAI_API_KEY` | VLM、反思、OpenAI 指令 JSON |
| `OPENAI_BASE_URL` | 兼容端点 |
| `OPENAI_VLM_MODEL` | 覆盖默认小模型名 |
| `VISION_CONFIG` | 自定义 `enhanced_detection_config.json` 绝对路径 |
| `USE_CUDA_CLIP` | `1`/`true`/`yes` 时 CLIP 尝试 CUDA |
| `VLA_TEST_INSTRUCTION` | `test_pipeline.py` 默认指令覆盖 |
| `EMBODIED_WHISPER_FORCE_CPU` / `EMBODIED_WHISPER_STRICT_CUDA` | Whisper 设备与回退策略 |

## 5. 快速开始

### 5.1 进入工程目录

```powershell
cd "d:\personal project\code\MyRobotProject"
```

（请按本机路径修改。）

### 5.2 冒烟测试

```powershell
python test_pipeline.py
```

期望三行：`[Perception] JSON` → `[Brain] Decision` → `[Motor] Tensor Shape`。未装 torch/CLIP 链时最后一行可能为 `unavailable`。

### 5.3 语音 CLI（无 GUI）

```powershell
python embodied_command_hub.py --cli --device cpu --asr-lang auto --instruction-backend openai
```

`--instruction-backend`：`openai` | `rules`。`--disable-instruction-cli` 仅 ASR。

### 5.4 代码串联示例

```python
from perception_engine import ManiSkillPerceptionWrapper, format_reference_library_for_vlm
from cognitive_brain import CognitiveBrain, class_for_id
from motor_adapter import build_semantic_phrase_for_film, encode_text_emb_for_film

wrapper = ManiSkillPerceptionWrapper()
# wrapper.perceive(...); scene = wrapper._format_for_vlm()

brain = CognitiveBrain()
decision = brain.decide("grasp the apple", scene_json)

phrase = build_semantic_phrase_for_film(
    decision["action"], decision["target_id"], class_for_id(scene_json, decision["target_id"])
)
text_emb = encode_text_emb_for_film(phrase, device="cpu")
```
