# 外部来源与依赖说明

本文件标注 **MyRobotProject** 内**非你独立从零编写**、或**自其它仓库/项目迁入**的组成部分，便于合规与排障。若与上游许可证冲突，请以上游仓库为准。

---

## 1. 完整子目录（第三方或课题组工程）

| 路径 | 说明 | 本工程中的用途 |
|------|------|----------------|
| **`graduate_pro/`** | 课题组/学长的 ROS2 与视觉工程树（含 `vision_ai` 包、`EnhancedDetectionPipeline`、YOLO/SAM2 配置与权重路径等） | `perception_engine.py` 通过 `sys.path` 接入 `vision_ai.detection.enhanced_detection_pipeline`，做检测与 3D 锚定 |
| **`ADM_DP/`** | 扩散策略（Diffusion Policy）相关代码树；内含 `policy/Diffusion-Policy` | `motor_adapter.py` 将 `ADM_DP/policy/Diffusion-Policy` 加入 `sys.path`，调用 `diffusion_policy.model.vision.language_film.LanguageFiLM` 与 CLIP 生成文本嵌入 |

> 上述目录通常带有各自的 **README、LICENSE 或 package.xml**。请勿当作本人独占版权的原创模块对外再发布而不保留上游声明。

---

## 2. Python 包与模型（通过 pip/conda 安装，不在仓库内）

| 组件 | 典型用途 |
|------|----------|
| **OpenAI CLIP**（`pip install git+https://github.com/openai/CLIP.git`） | `LanguageFiLM.encode_texts` 侧文本编码 |
| **faster-whisper** / **ctranslate2** | `cognitive_brain.LocalSTTCommander` 本地语音转写 |
| **openai** SDK | `parse_instruction_with_vlm`、反思、`embodied_command_hub` 的 Chat Completions |
| **ultralytics**、**Open3D**、**PyTorch** 等 | 由 `graduate_pro` / `vision_ai` 检测栈间接依赖 |

---

## 3. 本目录根部的自研/拼装模块（职责边界）

以下文件为当前联调栈的**分层封装**（逻辑自写，但**调用**上述外部栈）：

- `perception_engine.py` — 封装 `EnhancedDetectionPipeline`，输出场景 JSON  
- `cognitive_brain.py` — Whisper + OpenAI VLM + 反思  
- `motor_adapter.py` — 拼装语义短句并调用 **ADM_DP 内 LanguageFiLM**  
- `embodied_command_hub.py` — 无 GUI 语音 CLI + httpx 指令 JSON  
- `test_pipeline.py` — 冒烟测试  
- `perception_wrapper.py` / `vlm_brain.py` — 兼容转导至上述模块
